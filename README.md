cabs
====

Cab Management System

###

> Employee
    # Login
    # Book a cab (add/update)
    # View booked cabs
    # Cancel / Delete booked cabs
    
> Employee (Manager)
    # Login
    # View all the booked cabs by their employee
    # Approve / Reject cabs
    
> Cab Manager
    # Login
    # View all the booked cabs by all employees
    # Approve / Reject cabs
    
> Admin
    # Manage users (add/update/delete)
